# 第2回 演習ノート（受講者向け）

ブラウザだけで進められる画像処理の演習です。**読む・書く・結果を見る**が同じ画面で完結します。

## はじめかた

1. ブラウザで **`http://dx-pi-XX.local:8888/lab?token=dx2026`** を開く
   （`XX` は自分の座席の番号。開けないときは掲示の固定IP `http://<IP>:8888/lab?token=dx2026`）
2. 左の一覧から **`00_はじめに_環境確認.ipynb`** を開く
3. セルを選んで **Shift + Enter** で実行。上から順に進める

## ノートの順番

| 順 | ノート | 内容 | 対応セッション |
|---|---|---|---|
| 1 | `00_はじめに_環境確認.ipynb` | ライブラリ確認・はじめての1枚 | S1 |
| 2 | `01_画像の基礎.ipynb` | 画像＝数値配列・BGR・色空間 | S2 |

| 4 | `03_カメラとリアルタイム.ipynb` | ライブ映像・カラートラッキング | S4 |
| 5 | `04_AIカメラ.ipynb` | IMX500 で物体検出 | S5 |
| 6 | `05_統合_OpenCVとAI.ipynb` | AI＋OpenCV の小アプリ | S5–S6 |
| 7 | `06_グループワーク.ipynb` | モデル差し替え・アイデアソン足場 | S6 |

## コツ

- 緑の **✅ チェックポイント** が出たら次へ。
- カメラのセルでエラーが出たら、**前のノートの「後始末（`picam.close()`）」** を実行し忘れていないか確認。直らなければ **Kernel → Restart**。
- スライダーが出ないときも **Kernel → Restart** 後に上から実行。
- 詰まったら TA に挙手。AI に聞くときは **エラー全文** を貼る。

- まとめて取る場合は自分のPCから `scp "pi@dx-pi-XX.local:~/dx-session02/notebooks/*.jpg" .`
